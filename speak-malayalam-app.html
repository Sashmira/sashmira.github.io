Install on Android

Open this page in Chrome.
Use the Chrome browser on your Android phone.
Tap Install app or Add to Home screen.
Chrome may show an install prompt. If not, open the three-dot menu.
Launch SpeakMalayalam from your phone.
The app opens full-screen and keeps key pages fast.

Install on iPhone or iPad

Open this page in Safari.
Apple requires Safari for home-screen web apps.
Tap Share, then Add to Home Screen.
Name it SpeakMalayalam and tap Add.
Open from the home screen.
You get a clean app-style experience without searching each time.
